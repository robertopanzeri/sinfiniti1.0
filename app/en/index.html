We are a team of professionals dealing for more than twenty years with the Design and Re-design of digital tools.

What we do

Web Sites large scale

Web Apps

REST APIs

Design digital tools based on user needs and in close collaboration with the client, considering possible integrations with existing internal or external infrastructures and related implementation.

Deliver to programmers implementing the tool: the design documentation, an HTML/CSS user interface and the software specifications and architecture. Alternatively, we can implement the tool in collaboration with our technical partners.

Ensure the support and quality control during the implementation phase.

Working phases

We work by phases, and during the life cycle of the project they are carried out in sequence. Each phase can also play an autonomous role and be implemented independently.

Strategy

Analysis and concept

  • Analysis of business processes and user needs
  • Analysis of the integration of the digital tools with the company processes
  • Definition of functional requirements: operational and knowledge based
  • Concept and project guidelines

Design

Design of the digital tool

  • Interaction design for Web Applications: data model, workflows, page types
  • Information Architecture for Web Sites: content structure, navigation patterns, page types
  • Multi device User Interface design
  • Software specifications and architecture

Implementation

User Interface development

  • Modular layout systems (grids and page components)
  • Template development in HTML/CSS and JavaScript
  • HTML prototypes and documentation for the technological implementation

System components development

  • Database architecture: Transactional and Data warehouse
  • Middleware
  • Back-end development
  • System integration
  • API Definition and publishing
