How we work

Our working methodology unfolds in three areas developed in sequence: Strategy, Design and Implementation. Each one of these has a clear list of dedicated tasks. Based on client needs we can provide our service for all three areas, for just one area, or we can focus on a single activity within one area. The objective is to provide a tailor-made solution to your needs.

Strategy

Analysis and concept

  • Analysis and mapping of business processes and user needs
  • Analysis of integration needs of digital tools and connected business processes
  • Definition of operational and knowledge based requirements
  • Concept and project guidelines

Design

Digital tool design

  • Interaction design for Web Applications: data model, workflows, page types
  • Information architecture for Web Sites: content structure, navigation patterns, page types
  • Multi-device user interface design
  • Software architecture and specifications

Implementation

User interface development

  • Modular layout systems (grids and page components)
  • HTML/CSS and JavaScript template programming
  • HTML prototypes and implementation documentation
